# Workshop 2

This directory contains the design and modeling artifacts for the Workshop-2 assignment.


Included artifacts
- CRC (Class-Responsibility-Collaborator) cards — captures classes, responsibilities, and collaborators.
- Mockups — UI sketches and screens for the system flow.
- BPMN diagram — business process model and workflow for key scenarios.
- Architecture diagram — high-level system components and interactions.
- Class diagram — static structure, classes, attributes, and relationships.
- Relational database model — tables, keys, and relationships for persistent storage.


Purpose:
These artifacts document the system design and support implementation, review, and further refinement.
